# Manual de Usuario — StatCompare Sequito 📖

## Comandos Rápidos
- /sc o /statcompare: Abre el menú de configuración.
- /sc reset: Restablece la posición del panel.

## Uso en Raids
StatCompare es vital para verificar que tienes el **Hit Cap** necesario antes de un boss. Consulta el panel lateral al recibir buffs de raid (ej. Totems, Paladin Auras) para ver tu poder real.

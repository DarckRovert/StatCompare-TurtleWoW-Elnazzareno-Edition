# API Guide — StatCompare Sequito 🛠️

### StatCompare:GetStat(statName)
Retorna el valor total calculado para una estadística específica (ej. 'SPELLDAM').

### StatCompare:ScanUnit(unit)
Inicia un escaneo de estadísticas de la unidad especificada (ej. 'target' al inspeccionar).

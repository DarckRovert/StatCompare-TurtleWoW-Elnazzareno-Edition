# Changelog — Advanced Stats Visualization (StatCompare) 📈📊

## [v9.4.0] — 2026-04-07 [Omni-Tier]

### 🧪 Stat Scanner & Visualization (Mejoras)
- **Neural Stat Core**: Motor de escaneo de equipo reescrito para detectar las estadísticas exclusivas de Turtle WoW (ej: Bonos de daño de nuevas razas).
- **BCS Native Sync**: Sincronización milimétrica con `BetterCharacterStats` para evitar discrepancias de crítico y hit.
- **Apex Profile Integration**: Diseño de perfiles de estadística armonizado con pfUI para una estética institucional.
- **Inspect-Speed Booster**: Optimización del tiempo de carga al inspeccionar a otros jugadores en la raid.

### ⚡ Core & Branding
- **Rebranding [Omni-Tier]**: Elevación global de la versión y URLs institucionales a `ko-fi.com/darckrovert`.
- **SavedVariables Cleanup**: Gestión eficiente de perfiles de equipo guardados para personajes de bajo nivel.

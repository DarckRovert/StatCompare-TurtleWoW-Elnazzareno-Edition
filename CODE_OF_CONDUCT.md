# Contributor Covenant Code of Conduct

Como miembros de **El Séquito del Terror**, nos comprometemos a mantener la precisión en el análisis de estadísticas y un ambiente de colaboración profesional.

## Compromiso
- Veracidad en el reporte de bugs de cálculo.
- Respeto a las metodologías de theorycrafting del clan.
- Colaboración técnica para optimizar el escaneo de bonos.
